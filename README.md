# UniqueTab Chrome 拡張機能

ブックマークに登録されたサイトで同一 URL のタブが複数開かれることを防ぐ Chrome 拡張機能です。

## 機能

- 新しいタブで既に開いているタブと同じ URL を開こうとした際、既存のタブに自動的に遷移
- 新しいタブを削除することで、タブの重複を防止
- ブックマークに登録されている URL のみが対象
- 拡張機能のオン・オフ切り替え可能

## インストール方法

1. このリポジトリをクローン
2. Chrome で `chrome://extensions/` を開く
3. 「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このプロジェクトのディレクトリを選択

## 開発

### 必要な権限

- `tabs`: タブの監視と管理
- `bookmarks`: ブックマークの検索と判定
- `storage`: 設定の永続化
