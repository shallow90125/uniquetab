# テスト実行ガイド

## 単体テスト

### 前提条件

Node.js と npm がインストールされている必要があります。

### セットアップ

```bash
# 依存関係をインストール
npm install
```

### テストの実行

```bash
# 全テストを実行
npm test

# ウォッチモードで実行（開発中）
npm run test:watch

# カバレッジ付きで実行
npm run test:coverage
```

## E2E テスト（手動テスト）

Chrome 拡張機能の実際の動作を確認するための手動テストガイドです。

### 準備

1. Chrome で `chrome://extensions/` を開く
2. 「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. プロジェクトのルートディレクトリを選択

### テストケース

#### 1. 基本的なタブ統合機能

**前提条件:**

- 拡張機能が有効
- `https://example.com` がブックマークに登録されている

**手順:**

1. 新しいタブで `https://example.com` を開く（タブ A）
2. 別の新しいタブで同じ `https://example.com` を開く（タブ B）

**期待結果:**

- タブ B が自動的に閉じられる
- タブ A がアクティブになる
- タブ A のウィンドウが前面に表示される

#### 2. ブックマークされていない URL の動作

**前提条件:**

- 拡張機能が有効
- `https://test.example.com` がブックマークに登録されていない

**手順:**

1. 新しいタブで `https://test.example.com` を開く（タブ A）
2. 別の新しいタブで同じ `https://test.example.com` を開く（タブ B）

**期待結果:**

- 両方のタブが開いたまま維持される
- タブの統合は行われない

#### 3. 拡張機能の有効/無効切り替え

**手順:**

1. 拡張機能アイコンをクリックしてポップアップを開く
2. トグルスイッチを OFF にする
3. ブックマークされた URL で新しいタブを複数開く

**期待結果:**

- タブの統合が行われない
- 通常のブラウザ動作が維持される

**手順（続き）:** 4. トグルスイッチを ON に戻す 5. ブックマークされた URL で新しいタブを開く

**期待結果:**

- タブの統合が再び機能する

#### 4. 統計情報の表示

**手順:**

1. ブックマークされた URL で複数回タブ統合を実行
2. 拡張機能アイコンをクリックしてポップアップを開く

**期待結果:**

- 「統合されたタブ数」が正しくカウントされている
- 「最終統合時刻」が表示されている

**手順（続き）:** 3. 「統計をリセット」ボタンをクリック 4. 確認ダイアログで「OK」をクリック

**期待結果:**

- 統計情報がリセットされる（0 に戻る）

#### 5. ブックマーク変更の反映

**手順:**

1. `https://newsite.com` を新しいタブで 2 回開く
2. 両方のタブが維持されることを確認
3. `https://newsite.com` をブックマークに追加
4. 既存のタブを閉じる
5. `https://newsite.com` を新しいタブで 2 回開く

**期待結果:**

- ステップ 5 でタブの統合が行われる
- ブックマーク追加が即座に反映される

#### 6. 複数ウィンドウでの動作

**手順:**

1. 新しい Chrome ウィンドウを開く
2. ウィンドウ A でブックマークされた URL を開く
3. ウィンドウ B で同じ URL を開く

**期待結果:**

- ウィンドウ A がアクティブになる
- ウィンドウ B の新しいタブが閉じられる

#### 7. パフォーマンステスト

**手順:**

1. ブラウザのコンソールを開く（F12）
2. タブの統合操作を実行
3. コンソールログで処理時間を確認

**期待結果:**

- タブ統合処理が 100ms 以内に完了
- ブックマーク判定が 50ms 以内に完了
- ログに処理時間が表示される

#### 8. エラーケースのテスト

**ケース 1: 無効な URL**

- `chrome://settings` などの内部 URL を開く
- 期待: 統合処理がスキップされる

**ケース 2: 権限エラー**

- 拡張機能の権限を一時的に無効化
- 期待: グレースフルに動作が停止する
